# 뉴스레터 작성 가이드

## 📝 기본 작성 방법

관리자 페이지(`/admin`)에서 일반 텍스트로 작성하면 자동으로 HTML로 변환됩니다.

---

## 🎨 문법 가이드

### 1. 제목

```
## 대제목
### 소제목
```

**결과:**

- `## 텍스트` → 큰 제목 (H2)
- `### 텍스트` → 중간 제목 (H3)

---

### 2. 목록

```
- 항목 1
- 항목 2
- 항목 3
```

**결과:**

- 자동으로 불릿 포인트 목록 생성

---

### 3. 이미지 (핵심!)

```
![이미지 설명](https://이미지URL.com/image.jpg)
```

**결과:**

- 본문에 이미지가 삽입됩니다
- 여러 개 추가 가능!

**예시:**

```
## Web3 산업 동향

최신 트렌드를 소개합니다.

![Web3 블록체인](https://images.unsplash.com/photo-1639762681485-074b7f938ba0?w=800)

### 주요 포인트

- DeFi 성장
- NFT 확산
- 메타버스 발전

![NFT 시장](https://images.unsplash.com/photo-1620321023374-d1a68fbc720d?w=800)

결론적으로 Web3는 계속 성장할 것입니다.
```

---

### 4. 일반 문단

```
그냥 텍스트를 입력하면 자동으로 문단이 됩니다.

빈 줄로 구분하면 새로운 문단이 시작됩니다.
```

---

## 🖼️ 이미지 찾기

### Unsplash (추천)

1. https://unsplash.com 접속
2. 검색 (예: "blockchain", "technology", "business")
3. 이미지 우클릭 → "이미지 주소 복사"
4. 복사한 URL 사용

**팁:** URL 끝에 `?w=800&q=80` 추가하면 최적화됨

### 예시 URL:

```
https://images.unsplash.com/photo-1639762681485-074b7f938ba0?w=800&q=80
```
